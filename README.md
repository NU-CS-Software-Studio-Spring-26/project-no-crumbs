# No Crumbs

## Description

This app will allow friends to connect and eat good food together. It serves two primary purposes:

1. A user can post about a meal they would like to cook. Friends can sign up to join that meal.
2. A user can post about the leftovers they have from a meal. Friends can sign up to meet up and eat together, or pick up a meal.

## MVP

Our MVP must allow for users to have a profile, add other users as friends, and allow people to post and respond to posts.

## Functions

1. Sign-up/Login
2. Friend list, request, accept/decline
3. Posts/dashboard view
4. Create/edit a post
5. Respond/RSVP/comment on a post
